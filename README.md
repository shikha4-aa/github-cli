## Setup & Installation

### 1. Clone the repository
```bash
git clone https://github.com/shikha4-aa/github-cli.git
```

### 2. Move into the project folder
```bash
cd github-cli
```

### 3. Run the script
```bash
node index.js <github-username>
```

Example:
```bash
node index.js kamranahmedse
```
